# Trabalho Prático 2 - Grafo do Cavalo (3x3)

Projeto em Python estruturado no estilo algs4.

## Estrutura

```text
 t2-cavalo-algs4/
 ├── README.md
 ├── dados/
 │   └── entrada.txt
 └── src/
     ├── main.py
     ├── graph.py
     ├── cc.py
     ├── cycle.py
     └── algs4/
         ├── __init__.py
         └── bag.py
```

## Executar

```bash
python src/main.py dados/entrada.txt
```
